<p>Public class to hold the state of a zone. It contains the main state of a zone, like selected source, the current power status (like on or off) and the current volume level. It holds also additional data about the zone state, like 'Lats Update' to indicate the date and time when the last comamnd (=update) arrived from hardware device. It holds also the zone quality, idicating the quality of the data for this zone. See <see cref="ZoneQuality"/> for more information.<br/></p><p><br/></p>
