<p>This class implements the interface of the configuration service. <see cref="IConfigure"/> and <see cref="IConfigureInternal"/> The configuration service is a WCF-service, hosted as singleton. It defines functionality to read the actual configuration of the NuvoControl system. It defines functionality to modify the actual configuration of the NuvoControl system. It defines functionality to save the actual configuration of the NuvoControl system.<br/></p>
